## 🎯 About

**EMI Tesseract** is a powerful, user-friendly web application designed to help users calculate and track their loan EMIs (Equated Monthly Installments) with advanced prepayment planning capabilities. Whether you're managing a home loan, car loan, or personal loan, EMI Tesseract provides comprehensive insights into your loan repayment journey.

### Why EMI Tesseract?

- 🎨 **Beautiful UI**: Modern, responsive design that works on all devices
- 📊 **Visual Analytics**: Interactive charts powered by Chart.js
- 💡 **Smart Calculations**: Real-time EMI calculations with accurate interest breakdowns
- 📅 **Prepayment Planning**: Plan and track multiple prepayment strategies
- 🔍 **Detailed Insights**: Monthly schedules, annual summaries, and comparison views
- 💾 **Track Progress**: Mark paid EMIs and see your remaining balance
- 🆓 **100% Free**: No registration, no ads, no hidden costs

---

## ✨ Features

### Core Functionality
- ✅ **EMI Calculator**: Calculate monthly EMI for any loan amount, interest rate, and tenure
- ✅ **Flexible Tenure**: Input tenure in months, quarters, or years
- ✅ **Interest Breakdown**: See how much goes to principal vs interest each month
- ✅ **Total Cost Analysis**: Calculate total interest paid over loan lifetime

### Advanced Features
- 🔄 **Recurring Prepayments**: Set up automatic prepayment schedules
- 📈 **Multiple Prepayment Strategies**: Add and compare different prepayment scenarios
- 💰 **Savings Calculator**: See exactly how much you save with prepayments
- 📉 **Tenure Reduction**: Track how prepayments reduce your loan tenure
- ✅ **Payment Tracking**: Mark EMIs as paid to track your progress

### Visualizations
- 📊 **Balance Chart**: Visual representation of loan balance over time
- 💹 **Interest Chart**: Cumulative interest paid visualization
- 📈 **EMI Breakdown Chart**: Principal vs interest component per EMI
- 📅 **Annual Summary Chart**: Yearly payment breakdown
- 🔍 **Comparison Views**: Side-by-side comparison of scenarios

### Reports & Tables
- 📋 **Monthly Schedule**: Detailed month-by-month payment schedule
- 📆 **Annual Summary**: Year-wise payment and interest summary
- ⚖️ **Comparison Table**: Compare scenarios with and without prepayments
- 🔢 **Pagination**: Easy navigation through large datasets

## 🏁 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Suchitan10/EMI-Tesseract.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd EMI-Tesseract
   ```

3. **Open in browser**
   ```bash
   # Simply open index.html in your browser
   # Or use a local server:
   python -m http.server 8000
   # Then visit http://localhost:8000
   ```

## 💻 Usage

### Basic Loan Calculation

```javascript
// Example: Home Loan
Loan Amount: ₹50,00,000
Interest Rate: 8.5% per annum
Tenure: 20 years

Result: 
Monthly EMI: ₹43,391
Total Interest: ₹54,13,840
Total Payment: ₹1,04,13,840
```

### Adding Prepayments

```javascript
// Example: Annual Prepayment
Prepayment Amount: ₹1,00,000
Start From: Month 12
Frequency: Every 12 months
Number of Prepayments: Unlimited

Result:
Interest Saved: ₹15,20,000
Tenure Reduced: 5 years 3 months
```

### Tracking Paid EMIs

1. Enter number of months already paid
2. Click "Mark as Paid"
3. See remaining balance and tenure
